Wilson Knowledge Base 是个人知识管理系统的实践。核心流程：Agent 抓取感兴趣的文章 → 自动总结为结构化笔记 → Quartz v4 构建静态站点 → 部署到 Vercel（中国可访问）。

特点：全局自定义字体（无 Google Fonts 依赖）、全文 RSS、邮件订阅集成、自定义侧边栏导航、移动端优化。所有笔记按用户设定顺序排列，支持中英双语。
