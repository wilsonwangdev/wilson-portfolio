选型困难症的救星。Frontend Tools Explorer 提供交互式的工具对比界面，覆盖 bundler、test framework、state management、CSS solution 等类别。

每个对比维度包含：性能基准、DX 评分、社区活跃度、适用场景建议。数据驱动，定期更新。
