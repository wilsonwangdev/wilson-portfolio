# Wilson's Portfolio

Personal portfolio and project showcase — [www.wilsonhandbook.online](https://www.wilsonhandbook.online)

Built with [Astro](https://astro.build), [Tailwind CSS](https://tailwindcss.com), and [SolidJS](https://www.solidjs.com). Based on the [Astro Sphere](https://github.com/markhorn-dev/astro-sphere) theme.

## Features

- i18n — English (`/en/`) and Chinese (`/zh/`)
- Light / dark theme
- Full-text search (Fuse.js)
- Responsive
- RSS feed & sitemap

## Commands

| Command         | Action                        |
| :-------------- | :---------------------------- |
| `pnpm dev`      | Start dev server              |
| `pnpm build`    | Build to `dist/`              |
| `pnpm preview`  | Preview production build      |

## Deploy

Deployed on [Vercel](https://vercel.com) at [www.wilsonhandbook.online](https://www.wilsonhandbook.online).
