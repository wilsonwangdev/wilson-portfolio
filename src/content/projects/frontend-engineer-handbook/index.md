Frontend Engineer Handbook 是一份面向中级以上前端工程师的进阶指南。内容不止于 API 文档，更深入工程实践：构建工具链原理、性能优化方法论、常见架构模式、核心库源码解读。

站点基于 VitePress 构建，采用文档站组织方式，章节清晰，配有可运行的示例仓库。
