Agent Master Companion 是 Agent Master Handbook 的命令行伴侣，将手册中的概念转化为可直接使用的本地工具。支持任务模板、agent 配置管理、工作流编排等功能。

设计理念：CLI-first，零配置启动，与 Agent Master Handbook 文档站点深度联动。
